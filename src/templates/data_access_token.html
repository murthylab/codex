Provide FlyWire Data Access Token (aka Cave token)

Since FlyWire data is not public yet, we'll need to verify your access token. Tokens are available for anyone with FlyWire production access (if haven't got it yet, you can request here and come back once onboarded).

  • If you know your token just copy it into the box below.
  • If you've generated your token in the past but don't have it handy, copy it from here, then navigate back and paste it in the box below. Note that if you're not signed in to FlyWire, you may need to sign in with Google again - make sure to use the account that's registered with FlyWire.
  • If you need to create a new access token, use this link instead and follow the same instructions as above.
