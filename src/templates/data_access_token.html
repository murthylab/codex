Provide FlyWire Data Access Token (aka Cave token)

FlyWire data is not published yet, so to get access you'll need to provide your access token. Tokens are available for anyone who has access to FlyWire production data (if not, you can request access here and come back once you are in).

  • If you have your access token just copy it into the box below.
  • If you've generated your token in the past but don't have it handy, copy it from here, then navigate back and paste it in the box below (here's a sample screenshot for what to copy). Note that if you're not signed in to FlyWire in this browser, you may need to sign in with Google again - make sure to use the account that's registered with FlyWire.
  • If you need to create a new access token, use this link instead and follow the same instructions as above.
